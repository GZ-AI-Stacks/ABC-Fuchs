# ABC-Fuchs — Schreiben lernen für Kinder

Verkaufsseite und Demo-Version der Lern-App **ABC-Fuchs** von GZ-AI.-Stacks.

🌐 **Live:** https://www.gz-ai-stacks.de/ABC-Fuchs/

## Was ist das hier?

Eine Schreib-Lern-App für Kinder von Vorschule bis 4. Klasse. Schwungübungen, Buchstaben nachspuren, Wörter schreiben, freies Schreiben — alles in einer Single-File-App, die offline auf Handy, Tablet und PC läuft.

## Inhalt dieses Repos

- `index.html` — Verkaufsseite (Landingpage)
- `demo.html` — kostenlose Demo-Version (App ohne Speicher-Funktion)
- `impressum.html`, `agb.html`, `widerruf.html`, `datenschutz.html` — Rechtstexte
- `img/hero.png` — Hero-Bild
- `.nojekyll` — verhindert Jekyll-Verarbeitung durch GitHub-Pages

## Vollversion

Die Vollversion mit Speichern, allen Schriften und allen Modi wird über [Digistore24](https://www.digistore24.com) vertrieben und ist **nicht** Teil dieses Repos.
